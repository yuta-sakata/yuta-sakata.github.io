Hexo是一个小而美的博客框架,基于Node.js,简便小巧,本站自豪的使用Hexo驱动!

准备工作

这里会用到

  • Node.js
  • Hexo
  • git
  • Gitee

安装git

Git 是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目,具体安装过程请看这里

Node.js 安装

Node.js是一个事件驱动I/O服务端JavaScript环境，基于Google的V8引擎，V8引擎执行Javascript的速度非常快，性能非常好
前往Node.js官网，下载安装包点击这里 具体安装过程请看这里

使用npm安装Hexo

npm是在安装node.js时就已经安装完成了，所以你只需要在你的桌面 [Shift]+鼠标右键 选中 “打开PowerShell” ，接着输入命令并回车
npm -g install hexo

如果命令行提示如上图，就是安装完成了，就可以关闭命令行了

使用Hexo创建一个blog

选择一个盘或者文件夹，同样在那个位置使用 [Shift]+鼠标右键 选中 “打开PowerShell” 打开命令行执行
hexo init blog
等待命令执行完成，接着，你会在当前目录下看见一个名为blog的文件夹，那个就是你的博客啦

接着,进入blog文件夹,执行
hexo s
这是hexo的预览模式，这里它会创建一个在4000端口的服务，打开浏览器键入localhost:4000可以查看运行效果

在命令行按 Ctrl+C 可以暂停服务

撰写博客

在blog目录下，执行
hexo new "你的文章标题"
等待执行完成，这样，就创建好了一篇文章，目录在blog/source/_posts/下就能找到刚刚创建好的标题文件,接下来就是编写文章

撰写一篇文章


Hexo的文章编写是使用的markdown语法，所以用文本编辑器(如Vsual Studio Code) 打开刚刚创建的的文章 即可开始编写
